UWC-SupportCanvas

Author: del Rosario Ernesto

Overview

Simple websocket API for UWC-SupportCanvas. Inspired on the 'socketAPI' solution from Eldar Djafarov

Demo

Check please clients page demo and admin dashboard demo pages under the directory demo

Usage

For connecting support widget to any page you need to append these code to any place on your page. Just change path to backend based on nodeJS to correct one.

<script src="http://localhost:8001/src/app.js"></script>
<script type="text/javascript">
  $(document).ready(function () {
    wgt.setup({
      name: 'James Willson'
    });
  });
</script>

Also you need to have a toggler for chat box on your page. By the default we have an example which you can use.

<a href="#" id="supportWidgetToggler" data-support-action="open" data-html2canvas-ignore="true">
  Toggle support
</a>

You can create any unique toggle what you will like, but in this case you need to keep the data attributes on it data-support-action="open" and data-html2canvas-ignore="true"
